## MVP 原型

由于是单页 HTML 应用，当前即为可交互原型。

### 预览方式
直接在浏览器打开 `index.html` 即可体验完整流程：

1. **首页** → 输入姓氏
2. **预览页** → 查看起源（可点击解锁）
3. **报告页** → 完整家族故事
4. **起名页** → AI 生成名字

## 设计资源

### 配色方案
| 用途 | 颜色 | Hex |
|-----|------|-----|
| 主色渐变起点 | 紫蓝 | #667eea |
| 主色渐变终点 | 深紫 | #764ba2 |
| 强调色 | 珊瑚红 | #FF6B6B |
| 背景 | 浅灰 | #f8f9fa |
| 文字 | 深灰 | #333 |
| 文字辅助 | 中灰 | #666 |

### 字体
- 主字体：-apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC'
- 标题：28px Bold
- 正文：15-16px Regular

### 组件风格
- 圆角：12-16px
- 阴影：0 2-10px rgba(0,0,0,0.1)
- 按钮：渐变背景，圆角24px
