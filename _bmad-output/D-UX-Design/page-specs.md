# 阶段 4: UX Design - 详细 UX 规格说明

> **项目**：百家姓 AI 姓氏溯源助手  
> **阶段**：4 - UX Design  
> **产出**：Page Specs

---

## 1. 首页（Home）

### 布局结构
```
┌─────────────────────────────┐
│         顶部 Logo           │
├─────────────────────────────┤
│                             │
│      [姓氏输入框]           │
│        "请输入您的姓氏"      │
│                             │
│     [ 开始溯源 ] 按钮       │
│                             │
│   ──── 或 ─────            │
│                             │
│   [微信] [抖音] 一键登录     │
│                             │
├─────────────────────────────┤
│   🔥 已有 128,934 人溯源    │
└─────────────────────────────┘
```

### 交互规格
- 姓氏输入框：最多4个汉字，支持复姓
- 输入时实时校验，显示绿色边框
- "开始溯源"按钮：disabled 状态灰色，激活后渐变紫
- 第三方登录：微信/抖音图标，hover 时放大 1.1 倍

### 视觉规范
- 背景：渐变色（#667eea → #764ba2）
- 字体：思源黑体 Bold 32px
- 按钮：圆角 24px，阴影 0 4px 15px rgba(102,126,234,0.4)

```
│  │   姓氏起源故事       │   │

│  │                     │   │

│                             │
```

### 交互规格

- 解锁按钮：点击弹出支付弹窗
- 滚动时：顶部导航栏吸附

### 视觉规范
- 卡片背景：白色圆角 16px
- 锁图标：#FF6B6B 红色
- 按钮渐变：#667eea → #764ba2

```
│  └─────────────────────┘    │
│                             │
│  ┌─────────────────────┐    │
│  │   👤 历史名人       │    │
│  │   林则徐 / 林徽因   │    │
│  └─────────────────────┘    │

│  ┌─────────────────────┐    │
│  │   📋 家族辈分表     │    │
│  │   永/绍/继/承/守    │    │
│  └─────────────────────┘    │
│                             │
│   [ 🤖 AI 智能起名 ]        │
│                             │
│   [ 📤 分享到朋友圈 ]       │
│                             │
└─────────────────────────────┘
```

### 交互规格
- 卡片列表：垂直滚动，每卡片圆角 12px
- 迁徙地图：横向滚动，支持缩放
- 历史名人：点击展开详情
- AI 起名：点击跳转到起名页
- 分享：生成图片后弹出分享选择

### 视觉规范
- 卡片间距：16px
- 标题：18px Bold
- 正文：15px Regular
- 地图高度：200px

## 4. AI 起名建议页

### 布局结构
```
┌─────────────────────────────┐
│  ← 返回    AI 智能起名      │
├─────────────────────────────┤
│  宝宝性别：[男] [女]        │
│                             │
│  期望寓意：                  │
│  [输入关键词，如：聪明/健康] │
│                             │
│  家族辈分：永               │
│                             │
│   [ 🎲 生成名字 ]           │
│                             │
│  ─────── 推荐结果 ───────   │
│                             │
│  ┌─────────────────────┐    │
│  │ 🌟 林永哲            │    │
│  │ 寓意：智慧深邃       │    │
│  │ [收藏] [分享]        │    │
│  └─────────────────────┘    │
│                             │
│  ┌─────────────────────┐    │
│  │ 🌟 林永熙            │    │
│  │ 寓意：光明磊落       │    │
│  │ [收藏] [分享]        │    │
│  └─────────────────────┘    │
│                             │
└─────────────────────────────┘
```

### 交互规格
- 性别选择：单选按钮，选中填充颜色
- 关键词输入：多选标签 + 自定义输入
- 生成按钮：loading 状态 1.5s 动画
- 名字卡片：点击展开详细解释
- 收藏：本地存储 + 同步到"我的收藏"

### 视觉规范
- 选中状态：#667eea 背景白色字
- 标签：圆角 20px，hover 边框加深
- 推荐卡片：白色背景，左侧彩色边条
