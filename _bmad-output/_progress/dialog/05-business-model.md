# Step 5: 商业模式对话记录

## 开场问题

**Saga:** "谁为你这个产品付费？"

## 用户初始回应

Karl 给出非常清晰的商业模式架构：

### 核心结论

1. **谁为产品付费？**
   - 第一核心付费主体：C 端个人用户（分层付费，优先级绝对明确）
   - 第二增量付费主体：B 端企业客户（后期赋能，启动期绝不碰）
   - 绝对不做核心收入：乱序广告变现（碰了直接丢核心壁垒）

2. **产品形态优先级？**
   - 启动期：微信小程序（唯一核心载体）
   - 增长期：抖音 / 小红书小程序 + API 嵌入式服务
   - 成熟期：独立 APP + 全平台生态覆盖

### C 端分层付费设计

**红线 1：基础姓氏溯源功能永久免费** —— 拉新入口，建立信任
**红线 2：启动期完全不碰广告** —— 避免毁掉用户信任

#### 第一优先级：给孩子起名的年轻父母（启动期现金牛）

| 付费档位 | 客单价 | 核心产品内容 | 核心价值 |
|---------|--------|-------------|---------|
| 引流款 | 9.9 元 | 姓氏专属起名基础方案（10个带溯源解读的名字） | 极低决策成本，筛选精准用户 |
| 主力款 | 99-198 元 | 定制化起名全案（完整溯源报告+30个定制名字+姓氏家书） | 核心收入来源，独家「起名+寻根」一体化 |
| 溢价款 | 598-1998 元 | 家族传承起名定制（专家复核+家谱入谱预备+实体纪念册） | 高净值家庭，拉高客单价 |

#### 第二优先级：修家谱寻根的中年长辈（增长期第二曲线）

| 付费档位 | 客单价 | 核心产品内容 |
|---------|--------|-------------|
| 基础工具款 | 39-99 元 | 家谱 AI 整理工具（拍照识别、文言文翻译、电子家谱生成） |
| 主力服务款 | 198-598 元 | 专属寻根溯源报告（AI匹配史料、宗亲分支、迁徙路线） |
| 高客单全案 | 2000-20000 元 | 家族家谱编修全案（AI+专家联合服务，续修、印刷、宗亲会管理） |

#### 第三优先级：文化认同的年轻人（补充收入，传播放大器）

- 19.9 元/月轻会员：解锁完整迁徙地图、名人故事、壁纸模板
- 9.9 元单次定制：姓氏图腾壁纸、家族徽章、起源短视频

### B 端企业客户（成熟期补充）

**启动期绝对不碰**，潜在四类客户：
1. 母婴平台/品牌（宝宝树、妈妈网、月子中心）— 嵌入式服务、联名礼盒
2. 基因检测/家谱平台（Ancestry、家谱网）— API 技术输出、数据打通
3. 文化机构/文旅景区 — 定制化互动服务、文创联名
4. 线下服务机构 — SaaS 工具输出

### 产品形态路线图

| 阶段 | 时间 | 载体 | 核心功能 |
|------|------|------|---------|
| 启动期 | 0-6 个月 | 微信小程序 | 免费姓氏溯源 + 分层起名服务 |
| 增长期 | 6-18 个月 | 抖音/小红书小程序 + API | 公域流量破圈、轻量 B 端验证 |
| 成熟期 | 18 个月+ | 独立 APP + 全平台 | 复杂深度功能、高净值用户沉淀 |

### 为什么启动期只做微信小程序？

1. **获客成本天差地别**：点开即用，分享裂变门槛为 0
2. **开发迭代成本极低**：快速上线 MVP，快速试错
3. **支付闭环极短**：微信支付天然适配，3 步完成付费
4. **完全匹配用户场景**：准父母、长辈都是微信高频用户，家族群是精准流量池

## 最终商业闭环总结

> 用「免费姓氏溯源」拉新，用「起名付费」跑通从 0 到 1 的商业闭环，用「家谱/寻根付费」打造第二增长曲线、沉淀核心用户，用「B 端合作+品牌联名」做增量补充，全程靠增值服务变现，绝不靠广告毁掉核心心智，最终从一个起名入口，长成真正的国民级「家族根目录」。
